MERN Authentication System

A full-featured MERN stack authentication system with email OTP verification, login, logout, forgot password, and reset password functionalities.

Features

User registration (signup) with email OTP verification

User login with JWT authentication

Forgot password and reset password via OTP

Resend OTP functionality

Role-based authentication (e.g., customer, admin)

Secure password hashing with bcrypt

JWT stored in HttpOnly cookies

CORS support for local and production environments

Tech Stack

Frontend: React.js, Tailwind CSS

Backend: Node.js, Express.js

Database: MongoDB (Mongoose)

Authentication: JWT, bcrypt

Email: Nodemailer for OTP delivery
